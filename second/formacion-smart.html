Formación Smart

Live website

Summary

E-learning platform, oriented to teach about business and project management.
The project main focus was to enhance the usability, creating a more responsive version while taking into account the use of iframes to show interactive content.
Furthermore, was created a tutorial that guides first time users around the platform main features and contents to make it more user-friendly, and the loading time was reduced thanks to the refactoring made during the development process.

Technologies used

  • Javascript
  • Sass
  • Bootstrap
  • Twig
  • PHP
